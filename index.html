d = b++; alert(d); // 1
Присваивается предудущее число переменной, потом увеличение на 1, увеличенное число будет использовано при следующем вызове переменной

c = (2+ ++a); alert(c); // 5
Сначала увеличение a=a+1, а потом сложение 2+3, в конце присваивается c. Вывод "c".

Сложение как в предыдущем примере, но сложение 2 + 2 (переменная увеличена, но результат в нее будет записан только при следующем обращении)

alert(a); // 3
